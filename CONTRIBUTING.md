# Contributing

This repository is primarily a research reference. Contributions that add real findings, fix inaccurate information, or document new gaps and gotchas are welcome.

## What to Contribute

- **New gotchas** — if you hit a problem not documented here, add it to `docs/08-gotchas.md`
- **Gap updates** — if you solved a gap (especially G-05), document it
- **Corrected links** — NVIDIA documentation URLs change frequently
- **Additional resources** — useful videos, forum threads, or community guides

## How to Contribute

1. Open a GitHub Issue describing the finding
2. Submit a Pull Request with the change
3. Keep entries factual and based on direct experience — not speculation

## What Not to Contribute

- Fabricated step-by-step instructions not verified against the actual toolchain
- Content that duplicates the official NVIDIA documentation
